# Node Practice Project

This is a simple Node.js practice project built for learning backend development using Express.js and understanding how frontend and backend work together.

---

## 🚀 Features
- Basic Express server setup
- Routing in Node.js
- Static file serving (HTML, CSS, JS)
- Simple frontend + backend integration
- Beginner-friendly project structure

---

## ⚙️ How to Run This Project

### 1. Install dependencies
npm install

### 2. Start the server
node server.js

### 3. Open in browser
http://localhost:3000

---

## 🛠️ Technologies Used
- Node.js
- Express.js
- JavaScript
